Class: OrderedCollectionField

bookworm. OrderedCollectionField

The OrderedCollectionField class defines an API for collections the items of which are in a pre-defined order. Provides methods for accessing information about item order, as well as retrieving items based on order. TODO: Add managed index for fetching items by order.
